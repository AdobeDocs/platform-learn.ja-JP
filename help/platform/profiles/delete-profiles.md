# プロファイルの削除

リアルタイム顧客プロファイル API を使用してプロファイルストアからデータを削除する方法を説明します。 Profile API を使用すると、データレイクや ID グラフに影響を与えることなく、プロファイルストアからデータを削除できます。 これは、ID グラフの問題をトラブルシューティングし、少数のプロファイルにのみ影響するデータ取り込みでときどき発生するエラーを修正する場合に役立ちます。
