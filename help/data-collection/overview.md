# データ収集チュートリアル

データ収集は、Adobeが提供する次世代型の Web サイトタグおよびモバイル SDK の管理機能です。 データ収集は、顧客体験の実現に必要な分析、マーケティングおよび広告の各ソリューションをデプロイおよび管理するためのシンプルな手段を提供します。 これらのビデオとチュートリアルを使用して、この重要なテクノロジーを学ぶことができます。
