# データ収集チュートリアル

Data Collection は、Adobeが提供する次世代型の web サイトタグおよび Mobile SDK の管理機能です。 データ収集を使用すると、関連する顧客体験を強化するために必要な分析、マーケティング、広告などのソリューションを、簡単にデプロイして管理できます。 これらのビデオとチュートリアルを使用して、この重要なテクノロジーの学習に役立ててください。
