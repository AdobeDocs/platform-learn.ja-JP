# Insights：概要

キャンペーンに関する優れたインサイトを提供する多数のレポートを見てみましょう。  これらは例に過ぎません。  他にも多くの機能が標準で用意されており、レポートの作成や変更が簡単にできます。

覚えておく必要があるのは、プラットフォームに存在するすべてのフィールド（ネイティブで作成したフィールドや、カスタムフィールド）がレポートの一部になることです。  したがって、選択肢は事実上無限です。
