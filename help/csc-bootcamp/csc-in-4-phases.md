# コンテンツサプライチェーン（4 段階）

世界は今、デジタル化が第一です。 企業が折り合いをつけざるを得ず、すぐに消え去ることはないのが現実です。 このデジタル世界の一環として、私たちは直面しています [大きなコンテンツの急増](https://www.prnewswire.com/news-releases/content-marketing-market-size-to-grow-by-usd-487-24-billion--by-objective-platform-end-user-and-geography---forecast-and-analysis-2022-2026--301562808.html) より多くの人々がソーシャルメディアを閲覧しデジタル空間で時間を過ごしリモートで働くことで推進しています

これが意味するのは、コンテンツを配信するチャネル、市場、形式が多くなり、そのコンテンツを動的かつ最適化する必要性が高まるということです。

![85%のマーケターは、コンテンツをより迅速に作成して配信するよう求められていると言います。](./images/pressure-to-create-content.png)

_ソース：_ [_Deloitte Digital_](https://www2.deloitte.com/content/dam/Deloitte/uk/Documents/consultancy/deloitte-uk-future-of-experience-time-to-market.pdf)

良いニュースは、コンテンツの需要に追いつくことができるのと同じくらい難しいことで、コンテンツをより迅速かつ容易に計画、作成、配信する方法があることです。 これは、  **コンテンツサプライチェーン**. また、スムーズなプロセス、強力な戦略、堅牢なプラクティスを活用して、コンテンツのライフサイクル全体を合理化できる場合、

## コラボレーションは、コンテンツ・サプライ・チェーンをスムーズに行うための鍵となります。

コンテンツサプライチェーンとは、人、ツール、ワークストリームを統合して、コンテンツを効果的に計画、作成、管理、配信するプロセスです。 お客様がますます高速により多くのより優れたエクスペリエンスを要求するので、後ではなく、現時点で適切なコンテンツを得る必要があります。 これにより、サイロを分類してワークフローを統合することで、すべてのチームを圧迫し続けます。 また、テクノロジーへの投資からより多くの価値を引き出し、効率と価値を高めるのに役立ちます。

![コンテンツサプライチェーンに関与するフェーズとチームの概要。](./images/csc-overview.png)

理論的には、あらゆる場所の企業がコンテンツをより迅速に取り出すのに役立つプロセスです。 しかし、実際には、多くのサプライチェーンは機能をまたいだコンテンツのライフサイクルが自然にどのように機能しているかによって壊れています。 残念ながら、これらのチェーンが壊れると、内部のミスアライメントを生み出し、予算を削減し、従業員を減らし、市場に主要なメッセージを届ける速度を遅らせることができます。

### コンテンツサプライチェーンの欠点は何ですか？

- [**70%**](https://business.adobe.com/resources/reports/future-creative-experiences.html) コア以外の作業に費やされる時間を、役割をまたいで異なるシステムで手動のタスクを管理します。
- [**21%**](https://business.adobe.com/resources/reports/future-creative-experiences.html) コンテンツプロセス全体でボトルネックを特定したクリエイティブの数を把握することで、効果的なエクスペリエンスを作成するための課題となりました。
- [**31%**](https://www.fotoware.com/blog/dam-industry-trends-by-fotoware) クリエイティブやマーケターの最も難しいのは、複数の関係者とアセットを共有できないことです。

コンテンツサプライチェーンが壊れた背後にある一般的な犯罪をさらに詳しく見るには、次の 3 つの大まかなフェーズのそれぞれを見るのに役立ちます。

1. Adobe Analytics の  **計画**  フェーズ、リソースを追跡する一様なプロジェクトプロセスまたは中央の方法がない場合、複数の改訂、過度のコスト、コストの遅延、および作業のバランスを取ることができなくなります。 さらに、チームメンバー間の切断により、サイクルが無駄になり、優先順位の変更が遅くなります。
2. 期間  **実稼動** 、サイロ化されたシステムと切断されたワークフローは、ライブコラボレーション、アセットファインダビリティ、遅延、およびバーンアウトを制限します。 また、クリエイティブはタスクやアセットを手動で追跡し、繰り返しタスクを手動で実行する必要があるので、結果的にコストが増加し、エラーが増えます。
3. ～の時期  **配信** は、中央のアセット管理システムを備えておらず、コンテンツを拡張するための自動化も制限されているため、チャネルをまたいでまとまり、パーソナライズされたエクスペリエンスをチームが開始できなくなります。 さらに、断片化された配信ツールとデータは、アセットのパフォーマンスを正確に測定できるように制限します。

![サプライチェーンが壊れると、断片化されたシステムは混沌を生み出します。](./images/fragmented-supply-chain.png)

## コンテンツ・サプライ・チェーンを回避する時が来た

ミスステップやミスコミュニケーションに苦しむものと同様に、プロセスを改善する方法が常に存在します。 同じ 3 つの段階を踏まえて、お客様のビジネスにとって優れたコンテンツサプライチェーンがどのようなものかを見てみましょう。

### 合理化された計画により運用を最適化

マーケティングチームは、部門をまたいだコラボレーションを可能にし、プロジェクトを計画されたデータ主導型の成果に結び付ける作業管理ソリューションを使用するのが理想です。 また、マーケティングチームが作業を戦略的に優先し、クリエイティブチームにその優先度を知らせる力を与える必要があります。 そこから、クリエイティブの概要、コンテンツのバージョン、フィードバックを各プロジェクト内で結び付け、チームが古いメールを検索して探しているものを見つける必要がなくなります。

クリエイティブチームは、 [優れたデジタル資産管理](https://business.adobe.com/products/experience-manager/assets/digital-asset-management.html)[(DAM)](https://business.adobe.com/products/experience-manager/assets/digital-asset-management.html)[システム](https://business.adobe.com/products/experience-manager/assets/digital-asset-management.html) これは、クリエイティブおよび作業管理ツールと統合されています。 これにより、関連するアセットの検索、使用、更新、転用が簡単に行えます。 これらをすべて行うことで、市場投入のスピードを向上させ、社内の運用とプロセスをシンプル化できます。

### シームレスなワークフローにより、本番環境の効率を向上

計画を立てると、優れた実稼動フェーズは、作業管理アプリケーションと共に、すべてのシステム間でメタデータを同期する統合されたアセット管理ソリューションを中心に構築されます。 これにより、繰り返しの書式設定タスクを自動化し、品質を損なうことなく出力を高速化できます。 作業管理ソリューションがクリエイティブツールと統合されると、クリエイティブチームはリクエスト、ワークフロー、レビューを管理できるだけでなく、既に作業中のツール内でコンテンツを承認し最終処理することもできます。

その後、コンテンツワークフローに人工知能 (AI) を追加すると、顧客の関心に合わせて、すべての顧客の規模に応じて自動的にパーソナライズされたコンテンツを作成できます。 つまり、チームは全員が最善の行動をとることができます。クリエイティブは魅力的なコンテンツを構築でき、マーケターはインサイトを使用してそのコンテンツをカスタマイズできます。 素晴らしいエクスペリエンスを提供し始めるための準備をしました。

### アセットの配信とコンテンツ効率の測定

コンテンツを配信する際には、関係するすべての重要なユーザーやチームが、必要なコンテンツにアクセスできることが重要です。 DAM はニーズに合わせて構築され、チームがタイプ別、または選択した他のタグ別にアセットのカタログ化、追跡、検索をおこなうことができることを確認します。 顧客の動きは速く、チームはより速く移動し、ビデオ、インフォグラフィック、写真、メッセージなどをフラッシュで検索する必要があります。

これに加えて、真に優れたコンテンツサプライチェーンでは、画像のサイズ変更、バージョン管理、地域間での重複を回避できる DAM を使用します。 また、クリエイティブツール、作業管理ソリューション、コンテンツ管理システムなど、テクニカルスタック全体に統合して、チームが新しいコンテンツをすばやく簡単に作成して、任意のチャネルに配信できるようにする必要があります。 また、AI を使用すれば、顧客の位置、興味、行動に合わせて、関連するコンテンツを自動的に生成できます。

## Adobeでは、クラス最高のコンテンツ・サプライ・チェーンを構築しています

我々自身の [Adobe Experience Cloud事業](https://business.adobe.com/) は、以前に説明した手順に従って、キャンペーンを有効にし、複数のセグメントをサポートすることに重点を置いています。 当社のビジョンは、キャンペーンのフレームワークとメッセージに沿った、グローバルに魅力的なコンテンツを提供することでした。

エンドツーエンドのコンテンツ作成プロセスの鍵は、 [Adobe Workfront](https://business.adobe.com/products/workfront/main.html) および [Adobe Experience Manager Assets](https://business.adobe.com/products/experience-manager/assets/aem-assets.html). 同じ 3 つの高レベルの段階を踏まえて、プロセスにどのように取り組んでいるかを見てみましょう。

まずは、  **計画** ：何百ものコンテンツリクエストをビジネス全体から受け取り、四半期ごとの優先事項に基づいて優先順位を付けます。 その後、Workfrontのコンテンツ提案ダッシュボードを使用して透明性を高め、関係者と連携し、マーケティングチーム、クリエイティブチーム、Web チーム、セールスチーム全体にわたって迅速に承認を得ることができます。 これにより、四半期に対してコミットしている内容の記録を作成し、クリエイティブチームと Web チームに対して、リソース計画のための要求の総数を表示する機能を提供します。 さらに、管理者はチームのワークロードを簡単に監視できます。

我々が  **実稼動** 承認されたリクエストは、クリエイティブチームに詳細を提供するプロジェクトに変わります。例えば、クリエイティブブリーフ、Web ランディング場所、プロモーション要件などです。 その後、アドビのコンテンツストラテジストは、Workfrontの関係者やクリエイティブと協力して、コンテンツをレビューし、期限を満たします。 これらをWorkfrontで行うので、重複する手順や不要な E メール、複数のフォームの入力を排除し、アセットタイプ別に実稼動サイクルの追跡を開始しています。 これにより、スタッフのリソースを調整し、必要に応じて優先順位を再設定し、プロジェクトデータを詳細に把握できます。

コンテンツの準備が整ったら、  **配信** , we [オファー、アセット、プロモーションコンテンツを公開](https://business.adobe.com/customer-success-stories/adobe-content-hub-case-study.html) Experience Manager Assets 特に、すべては社内のマーケティングハブに送られ、社内の誰でも必要なコンテンツを、必要なときにすぐに見つけることができます。

コンテンツサプライチェーンの構築による大きな報酬を見てきました。 特に、1 つの作業管理システムへの移行により、重複データのエントリを半分に削減しました。 常に改善の余地があるにもかかわらず、お客様の期待を一歩先に進めるためのプロセスに進んでいることがわかります。
