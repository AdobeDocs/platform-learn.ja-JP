# 1.7 Adobe Commerce用のインテリジェント開発者ツール

## 学習内容

このモジュールでは、Cursor などのインテリジェントな開発者ツールを使用して、Adobe Commerce as a Cloud Service環境に対する拡張機能を開発します。 この拡張機能の目標は、着信注文イベントをサードパーティのエンドポイントに転送することです。 Adobe Commerce as a Cloud Serviceのイベント転送は、Adobe I/O App Builder、Adobe I/O EventsおよびAdobe I/O Runtimeに依存します。 これらすべてのサービスの設定は、Cursor によって支援されます。

## 前提条件

- Adobe Commerce as a Cloud Serviceにアクセスします。
- Adobe Experience Manager as a Cloud Serviceへのアクセス。
- 開発者またはシステム管理者のアクセス権を持つAdobe I/Oにアクセスします。
- カーソルへのアクセス。
- 依存関係として [Adobe Experience Manager Cloud ServiceとEdge Delivery Services](./../../../modules/asset-mgmt/module2.1/aemcs.md) があるので、このモジュールを完了している必要があります。
- 依存関係 [Adobe Commerce as a Cloud Service](./../../../modules/asset-mgmt/module1.5/accs.md) があるので、このモジュールを完了している必要があります。
