# 分析とエージェント

## 学習内容

このモジュールでは、アドビ以外の外部エージェントを使用して、Adobe AnalyticsやCustomer Journey Analyticsにあるデータを分析します。

## 前提条件

- Adobe Analytics（AA）およびAdobe Customer Journey Analytics（CJA）へのアクセス
- 有料サブスクリプションで ChatGPT や Cloud.ai にアクセス

## 演習

[1.5.1 MCP サーバーとCJAおよび ChatGPT](./ex1.md){target="_blank"}

この演習では、ChatGPT でカスタム MCP サーバーを設定して、ChatGPT をCustomer Journey Analyticsに直接接続し、Adobe Customer Journey Analytics（CJA）で直接 ChatGPT でホストされているデータの分析を開始します。

[1.5.2 MCP サーバを搭載したCJAおよび Claude.ai](./ex2.md){target="_blank"}

この演習では、Claude.ai にカスタム MCP サーバーを設定して、Claude.ai をCustomer Journey Analyticsに直接接続し、Adobe Customer Journey Analytics（CJA）でホストされているデータを Claude.ai から直接分析します。

[1.5.3 MCP サーバとAdobe Analyticsおよび ChatGPT](./ex3.md){target="_blank"}

この演習では、ChatGPT でカスタム MCP サーバーを設定して、ChatGPT をAdobe Analyticsに直接接続し、Adobe Analytics（AA）でホストされているデータを ChatGPT から直接分析します。
