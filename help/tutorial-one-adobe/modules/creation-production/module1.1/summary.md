# Firefly サービスの概要と利点

Firefly サービスは、ワークフローを自動化するジェネレーティブ AI およびクリエイティブ API の包括的なセットです。 これらの機能を任意の制作ワークフローまたはエクスペリエンスワークフローに組み込むことで、組織はコンテンツの生成、編集および組み立てを拡張し、品質と制御を維持しながらコンテンツの制作を自動化することができます。
